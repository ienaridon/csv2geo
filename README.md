## 事前準備
このプログラムは住所からの緯度経度変換にYahoo! Japan提供のYOLP APIを利用しています。  
プログラムからこのAPIを使用するためには、事前にYJDNに登録してAPPID(CLIENTID)を取得しておく必要がああります。
